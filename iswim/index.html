iSWIM — a secure and smart way for insurers to handle original documents.

LOGIN

Handling and storing original documents have always been a headache for insurers.

iSWIM provides a solution to handle original documents using state of the art technology — blockchain.

FEATURES
Storage

Original documents are scanned and kept in a safe place.

Copies

A QR code will be stamped on the “true copy” and passed to relevant parties.

Retrieval

Clients can retrieve copies or the original documents. All handling of documents is logged securely.

Browsing

Useful data can be extracted from the documents and passed to clients through an API or in formats like JSON or CSV.

WORKFLOW
Original documents are scanned and uploaded to our system with a reference and location code. Relevant data is extracted and updated in our system as well.
Clients will receive copies with QR codes pasted on the corner. Clients can scan the QR code to access information on the document.
Clients can request data on the document, ask for another copy, or request the originl document.
